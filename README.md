# LAN Transfer CLI

An ultra-fast and extremely simple command-line tool to transfer files directly between computers on your local network (LAN) using a P2P (Peer-to-Peer) connection.

Forget about configuring SSH ports, setting up FTP servers, or using USB flash drives. Expose your file with a single command, and download it with another.

## Features
- **Direct P2P:** Maximum transfer speed leveraging your local LAN.
- **No Complex Setups:** It just works. No databases or previous configurations required.
- **End-to-End Encryption (E2EE):** Military-grade AES-256-GCM encryption on the fly. Your file travels as pure noise, impossible to intercept.
- **PIN Authentication:** Random 4-digit PIN blocks unauthorized users from "stealing" the transfer.
- **"One-Time Use" Security:** As soon as the download finishes successfully, the mini-server shuts down automatically.
- **Beautiful Interface:** Built with `rich` for informative panels and `tqdm` for precise progress bars showing MB/s and time remaining.

## Installation

### For Windows (The Easiest Way)
Simply open the `dist/` folder and copy the **`lan-transfer.exe`** file to any folder on your computer or a USB drive. 
You don't need to install Python or configure anything! Just open your console and run the `.exe`.

*(If you modify the source code, double click `build.bat` to compile a new `.exe`).*

### For Linux / Developers (Source Code)
The project uses the modern `pyproject.toml` standard.
1. Clone or download this repository.
2. Open your terminal in the project folder and run:
```bash
make install
# or alternatively: pip install .
```

## Usage

The application has two main commands: `send` and `receive`. You can use `lan-transfer --help` at any time to see the internal documentation.

### 1. Send a file
Run the script in `send` mode pointing to the file you want to share.

```bash
lan-transfer send "/path/to/Video.mp4"
```
*The program will generate a security PIN, an Encryption Key, and the exact command for the receiver to download.*

### 2. Receive a file
On your other computer, simply paste the command generated by the sender:

```bash
lan-transfer receive 192.168.1.5:43210 "Video.mp4" --pin 1234 --key A8X2...
```
*You will immediately see a detailed progress bar. Once it reaches 100%, the file will be saved in your current directory decrypted, and the sender will shut down completely.*
